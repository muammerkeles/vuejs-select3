VueJs ve Bootstrap 4 kombinasyonu ile
`Ajax` özelliği olan "Autocomplete" gibi çalışan bir "Selectbox" komponent inşa ediliyor.

With the combination of VueJs and Bootstrap 4, a "Selectbox" component that works like "Autocomplete" with `Ajax` is being built.

Live Demo

Selected Data
{{secilenObjem.ajax}} 
Vue 

var app = new Vue({
  el: "#app1",
  data(){
    return{
                                                        ajaxsettings:{
	  url 			: 	"",
	  requestKey 		: 	"",
	  method 		: 	"",
	  dataRootKeyPattern 	: 	"",
	  displayKeys 		: 	[]
	}
  },
  methods:{
       MyData_ajax:function( dt){
       this.secilenObjem.ajax= dt;  // 'dt' is selected object data! (assigned copy)
       },
  }



HTML

<mkselect class="text-left" v-on:getselected="MyData_ajax"  v-bind:ajaxsettings= "myajax_settings"> </mkselect>
